1. Update Item Status

When item removed from the catalog, or maybe sold out. To make it possible disappear from recommendation results is remove it.

Protocol method REST API URL Params
DELETE /v2/items /{item_id} 
 curl -X DELETE "http://api-aws.predictry.com/v2/items/5205" \ 
 -H "Accept: application/json" \ 
 -H "X-Predictry-Server-Tenant-ID: $TENANT_ID" \ 
 -H "X-Predictry-Server-Api-Key: $API_KEY"

Sample response

{
   "error":false,
   "status":200,
   "message":"Item succesfully removed",
   "client_message":""
}
